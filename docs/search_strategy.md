# Search Strategy

This project tracks laser-driven ion acceleration broadly, not only a single acceleration mechanism. The query set is organized around these buckets:

## Core terms

- laser ion acceleration
- laser-driven ion acceleration
- laser proton acceleration
- laser-plasma ion acceleration
- laser-accelerated proton

## Acceleration mechanisms

- TNSA / target normal sheath acceleration
- RPA / radiation pressure acceleration
- hole-boring and light-sail regimes
- BOA / breakout afterburner
- relativistic transparency
- CSA / collisionless or electrostatic shock acceleration
- MVA / magnetic vortex acceleration
- near-critical-density target acceleration

## Theory and simulation

- PIC / particle-in-cell simulation
- hydrodynamic simulation
- scaling laws and parameter scans
- laser-target coupling and density-profile optimization

## Experiments, targets, and diagnostics

- high-repetition-rate operation
- target wheels and automated alignment
- nanostructured, microstructured, mass-limited, liquid-sheet, gas-jet, and near-critical targets
- beamlines, transport, dosimetry, Thomson parabola, radiochromic film

## Applications

- laser-driven neutron sources
- proton therapy and hadron therapy
- laser-FLASH and radiobiology
- materials and fundamental physics applications

## Machine learning and automation

- machine learning for proton or ion acceleration
- Bayesian optimization / Bayesian optimisation
- closed-loop optimization / optimisation
- active learning, surrogate models, neural networks, Gaussian processes

The ranking terms intentionally overlap with the query terms. Queries decide what gets fetched; ranking decides what gets promoted into the report.
